# fish shell config file

## install
```
$ brew install fish
```

## /etc/shells
```
+ /usr/local/bin/fish
```

## switch default shell to fish
```
$ chsh -s /usr/local/bin/fish
```

## set config
```
$ fish_config
```

## mv this file
```
~.config/fish/config.fish
```
